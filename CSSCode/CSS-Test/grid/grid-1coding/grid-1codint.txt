1. 그리드 구조

1-1. flex처럼 부모와 자식 관계로 이루어져 있다.
1-2. 부모 => grid container/ 자식 => grid items
1-3. 부모에 display : grid를 설정해줘야 작동한다.
1-4. flex와 마찬가지로 부모로부터 전체적인 설정/ itmes 각자의 설정으로 나뉜다.


2. grid 용어

2-1. grid-container
grid-item들이 영향을 받는 공간

2-2. grid-items
grid-container에 의해 영향을 받는 item

2-3. 그리드 트랙 (Grid Track)
Grid의 행(Row) 또는 열(Column)

2-4. 그리드 셀 (Grid Cell)
Grid의 한 칸. <div>같은 실제 html 요소는 그리드 아이템이고, 
이런 Grid 아이템 하나가 들어가는 “가상의 칸(틀)”

2-5. 그리드 라인(Grid Line)
Grid 셀을 구분하는 선

2-6. 그리드 번호(Grid Number)
Grid 라인의 각 번호입니다.

2-7. 그리드 갭(Grid Gap)
Grid 셀 사이의 간격

2-8. 그리드 영역(Grid Area)
Grid 라인으로 둘러싸인 사각형 영역으로, 그리드 셀의 집합


3. grid property
3-1. grid-template-rows
정의 : 행의 배치를 정의한다.


grid-template-columns
정의 : 열의 배치를 정의한다.
