1. box-sizing : content-box (default)
=> 요소의 총 넓이와 높이는 컨텐츠 + padding

2. box-sizing : border-box
=> 요소의 총 넓이와 높이는 width이 200px이라면 컨텐츠 + padding + border 모두 합쳐서 200px이라는 뜻입니다.  


3. min-width (기본값은 auto)
=> 요소가 얼마나 작아질 수 있는지 제한하는 속성입니다.
=> auto의 경우 컨텐츠 길이 만큼 넓이가 증가합니다. (grid shell 깨짐)
=> min-width : 0 으로 설정하면 딱 넓이 만큼만 차지합니다.
