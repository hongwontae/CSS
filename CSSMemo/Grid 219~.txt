1. 그리드는 자동으로 모든 자식 요소를 배치하고 왼쪽 상단에서 시작하여 오른쪽 하단 순으로 채워 넣는다.

2. 자식 요소에서 이를 바꿀 수 있다.

**그리드 라인 번호**
=> 그리드는 칼럼과 로우의 라인 번호를 인식한다. 이를 이용해서 속성의 값의 변화를 줄 수 있다.
=> grid 219에 레이아웃을 기준으로 그리드 칼럼 라인 번호는 요소의 왼쪽에 줄을 그으면 그것이 기준이 된다.
=> grid 219에 레이아웃을 기준으로 그리드 로우 라인 번호는 요소의 위에 줄을 그으면 그것이 기준이 된다.

2-1. grid-column-start:3, grid-column-start:5
라인3에서 라인5까지 해당 요소가 먹어라.
2-2. grid-row-start:1, grid column-end:3
라인 1에서 라인3까지 해당 요소가 먹어라

=> 예제를 보면서 이해하는 것이 더 쉽다.

3. 속성값 auto
=> auto를 넣으면 나머지 값이 다 계산된 나머지를 다 갖는다.

4. repeat(f,s)
=> f :  얼마나 자주 반복할지 
=>  s :  반복해야 하는 패턴

5. minmax(f,s)
=> f : 최소값/ 행의 경우 높이, 열의 경우 너비가 된다.
=> s : 최대값/ "",""
