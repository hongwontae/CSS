1. SASS란?
정의 : CSS 전처리기, CSS가 브라우저에 배포되기 전에 VANILA CSS로 변환된다.

=> SCSS는 SASS를 도와주는 기능을 의미한다. (.jsx와 비슷하다.)

특징
1-1. CSS 규칙을 중첩해서 코드를 짧게 만들 수 있다.
1-2. 다른 규칙을 상속할 수 있다.
1-3. 헬퍼 함수를 사용하여 원하는 스타일링
1-4. 믹스인을 통해 재사용할 수 있는 코드 조각 생성
1-5. 분할 파일로 코드를 여러 파일로 나눌 수 있다.
1-5. 변수 정의 가능
