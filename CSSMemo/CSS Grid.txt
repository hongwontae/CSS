1. 부모 요소 Grid Container/ 자식 요소 Grid  Item이라고 한다.

2. 부모 요소에 display:grid를 주면 적용된다. = Flex와 같다.

3. 그리드 트랙
=> grid의 행 또는 열

4. 그리드 셀
그리드 그리드 아이템이 들어가는 한 칸을 의미한다.

5. 그리드 라인
grid 셀을 구분하는 선

6. 그리드 번호 
grid 아이템의 번호

7. 그리드 갭
grid 셀 사이의 간격

8. 그리드 영역
grid 라인으로 둘러싸인 사각형 영역으로, 그리드 셀의 집합이다.

grid는 flex와 마찬가지로 컨테이너에 적용하는 속성, 아이템에 적용하는 속성 이렇게 둘로 나뉜다.


9. 그리드 형태 정의
컨테이너에 Grid 트랙의 크기들을 지정해주는 속성이다.
grid-template-rows는 행(row)의 배치 // 로우의 크기
grid-template-columns는 열(column)의 배치 // 칼럼의 크기

단위를 다양하게 구성하여 사용할 수 있다.
9-1. px

9-2. fr
=> 숫자 비율대로 트랙의 크기를 나눈다. px과 연동하여 사용할 수 있다.
=> auto 값을 부여하면 나머지 크기를 다 갖는다.

10. repeat()
=> 반복되는 값을 자동으로 처리하는 함수
=> repeat(반복횟수, 반복값)

11. minmax()
최소값과 최대값을 지정하는 함수이다.
ex) minmax(100px, auto) => 최소한 100px, 최대는 자동으로 늘어나게 해준다.

12. auto-fill, auto-fit
칼럼의 개수를 미리 정하지 않고 설정된 너비가 허용하는 한 최대한 셀을 채운다.
auto-fill => 엄격/ auto-fit는 자유로움

13. gap
그리드 셀 간격 설정
row-gap : row의 간격 설정
column-gap : column의 간격 설정
gap : 모든 간격 설정 (10px => row,column 같은 간격 설정) (10px 5px => row는 10px, column은 5px)

14. grid-auto-columns/ grid-auto-rows
통제를 벗어난 위치에 있는 트랙의 크기를 지정하는 속성이다.
