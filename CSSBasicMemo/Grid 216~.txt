1. display : grid
=> 부모 요소가 이 속성을 가지고 부모 속성 밑의 자식들은 grid cell이 된다.
=> 이 속성값은 하나의 열만 생성하고 자식 요소만큼 행을 생성한다. => 기본값
=> 직계 자식만 그리드의 셀이 된다/ 행을 생성하는 매커니즘은 직계 자식만 기능한다.
=> 기본값은 너비는 100%, 높이는 컨텐츠만큼 가진다.

2.  grid-template-columns
=> 부모 요소에서 사용가능하다.
=> 값은 각 열의 너비를 설정하면 된다. 값의 개수는 칼럼의 개수를 의미한다.

=> ex) 200px, 150px, 20%
=> 3개의 칼럼으로 변하고 각 요소들은 200px, 150px, 부모의 20%를 갖는다. 요소가 네 개면 하나는 자동으로 내려간다.

3. grid-templates-rows
=> 부모 요소에서 사용가능하다.
=> 값은 각 행의 높이를 설정한다. 값의 개수를 행의 개수를 의미한다.

ex) 5rem, 2.5rem
=> 1개의 행은 5rem, 2개의 행은 2.5rem 높이를 갖는다.

종합
=> 예시 container에 div가 5개 있다.
=> Columns는 너비 값을 지정해주면 된다. 지정해준 너비 값이 4개면 칼럼이 4개이다. 나머지 한 요소는 밑으로 내려가서 처음 지정해준 너비 값을 갖는다. 이 떄 div 요소처럼 높이는 컨텐츠만큼 갖는다.
=> 이에 rows의 값을 2개를 주면 높이를 갖는다. 한 칸의 칼럼들은 처음 로우의 높이 값을 갖고 나머지 하나의 요소는 내려가서 처음 colums+ 처음 rows의 값을 갖는다.
